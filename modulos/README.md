# Proyecto Fundación Banco Caja Social

## Descripción
Este proyecto contiene la codificación de módulos de software según los requerimientos planteados.

## Módulos desarrollados
- Figuras geométricas
- Análisis de edades
- Mezcla de vectores
- Registro de preferencias musicales

## Tecnologías
- JavaScript
- HTML
- CSS

## Ejecución
Abrir el archivo index.html en el navegador.
